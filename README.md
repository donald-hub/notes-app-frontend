# MERN Notes App

## Tech Stack
- MongoDB
- Express
- React
- Node.js

## Features Completed
- Create note
- Read all notes
- Read note by ID
- Update note
- Delete note

## API Routes
GET    /notes
GET    /notes/:id
POST   /notes
PUT    /notes/:id
DELETE /notes/:id

## Frontend Pages
- HomePage (list notes)
- EditNote (update note)
- CreatePage

## Services
- services/notes.js handles all API calls
